# Hospital Data Analysi

## 📌 Project Overview
SQL + Excel analysis on 100 hospital records across 10 Indian cities
to identify medical burden, cost efficiency and billing anomalies.

## 🛠️ Tools Used
- PostgreSQL (8 SQL queries)
- Microsoft Excel (Pivot Tables + Charts)

## 📊 Excel Analysis
- City-wise medical burden (patients + expenses)
- Monthly expense trend (line chart)
- Most efficient hospital
- Avg stay duration by department
- Patients vs expenses scatter chart

## 🗃️ SQL Analysis
- Q1: City-wise medical burden
- Q2: Long stay anomaly detection
- Q3: Department avg expenses
- Q4: High patient load cities
- Q5: Hospital benchmarking
- Q6: Critical patient zones
- Q7: Subquery — hospitals above avg
- Q8: Window function — expense ranking

## 🔍 Key Insights
- Ahmedabad = highest medical expenses (₹5.2L)
- Jaipur = most patients (1505) at lowest cost = best efficiency model
- Heritage Hospital = most cost efficient (₹158/patient)
- 46% expense gap between rank 1 and rank 10 hospital

- Hospital_Data_Insight.xlsx — Excel analysis with 5 charts
